# Banking Analytics Project

## Project Overview

This project analyzes customer, account, transaction, loan, card, and district data for a banking institution.

The objective is to transform raw banking data into actionable business insights using **SQL, Python, and Power BI**.

---

## Business Problem

The bank lacked centralized reporting for:

- Customer Analytics
- Transaction Analysis
- Loan Portfolio
- Card Performance
- District Performance

## Business Insights

- Majority of customers belong to the working-age population.
- Debit transactions exceed credit transactions.
- Loan default rate is approximately **6.6%**.
- Prague district has the highest customer concentration.
- Predictive analytics helps identify potential loan defaulters.
